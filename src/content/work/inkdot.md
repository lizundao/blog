前段时间把 [墨点](https://inkdot.cn/) 上线了。

站点地址是 [inkdot.cn](https://inkdot.cn/)，一句话介绍：**小工具，一点就够用**。它不是又一个「全能办公套件」，而是一组按需取用、装完就能用的桌面小工具，面向 macOS 和 Windows。

![墨点 inkdot.cn 首页截图](/images/sites/inkdot.png)

## 为什么要做墨点

日常工作中，我经常遇到这类场景：只想压缩几张 PDF、给图片批量加水印、清一下文件元数据，却不得不打开臃肿的软件，或者把文件上传到某个在线服务。

后者的问题很明显：网络、隐私、速度，都不在你手里。

墨点想解决的就是这件事：**把高频、轻量的操作做成独立小工具，本地完成，不折腾。**

## 放心用，不折腾

站上有一句标准叫「放心用，不折腾」。不是口号，而是每一款工具发布前都要过的 checklist：

| 标准 | 含义 |
|------|------|
| **免费** | 永久免费，无内购、无订阅；觉得好用可以自愿打赏 |
| **离线** | 本地运行，断网也能用，不依赖云端 |
| **安全** | 数据不出设备，不收集、不上传、不追踪 |
| **方便** | 即装即用，轻巧顺手，一点就启动 |
| **绿色** | 无广告、无捆绑、无后台骚扰 |

这五条看起来简单，做起来却要反复取舍。比如「方便」和「功能全」常常打架，我的做法是：**单款工具只做好一件事，需要组合能力时再出下一款。**

## 目前在架的工具

截至写这篇文章时，[墨点](https://inkdot.cn/) 上共有 3 款桌面工具，还在持续更新：

### PDF 工具箱

精选推荐款。压缩、合并、转换、水印、脱敏，PDF 相关的常见操作尽量在一个工具里本地搞定。文件不上传云端，适合处理合同、扫描件、需要脱敏的文档。

### 图片处理器

批量压缩、水印、格式转换、裁剪、旋转、合并、GIF 合成等，一共 14 种功能。适合整理素材、发文章前处理配图这类重复劳动。

### 元数据管理

查看、编辑、清除 PDF、Office、图片文件中的元数据。内置投标脱敏、隐私清除等预设，支持批量处理和前后对比。版本 v0.1.0，还在打磨细节。

更多工具会按实际需求陆续补上，不追求数量，只求每一款都「一点就够用」。
